# Sales-Performance-Dashboard-Power-BI
Interactive Power BI dashboard to analyze sales, profit, and regional performance trends with data-driven insights.
📌 Overview :
This project analyzes sales data to track revenue, profit, and regional performance trends.

📂 Dataset:
Sales dataset including product, region, revenue, and profit

🛠 Tools Used:
Power BI
Power Query
DAX

📈 Key Features:
Revenue and profit analysis
Regional and category-wise performance
KPIs: Total Sales, Profit, Growth Trends
Interactive filters and visualizations

🔍 Key Insights:
Identified top-performing products and regions
Highlighted underperforming areas
Analyzed monthly sales trends
